# dinosaur-settings-ui

《恐龙哪里跑》风格的 Q版卡通游戏设置界面，使用 React + Vite + JavaScript 构建。

## 本地运行

```bash
npm install
npm run dev
```

## 构建检查

```bash
npm run build
```

## 功能

- 750x1334 竖屏手机画布，桌面浏览器中居中等比预览。
- 游戏弹窗式设置界面，包含音效、音量、震动、士兵等级、等级数值设置项。
- 开关、音量滑杆、关闭/打开、恢复默认、保存提示均可交互。
- 纯 CSS 和 SVG 绘制视觉，不依赖复杂 UI 库。
